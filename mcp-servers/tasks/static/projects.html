Build rules the AIUI Agent will follow
  • Static HTML + Tailwind CDN + vanilla JS only (no build step).
  • Semantic HTML5: <header> / <main> / <footer>, one <h1>.
  • Mobile-first responsive, works from 320px up.
  • Accessibility: alt text, form labels, visible focus, contrast ≥4.5:1.
  • No Lorem ipsum — real copy from your description.
  • Whitelisted CDNs only (Tailwind, Google Fonts, jsDelivr, unpkg).
  • Every commit is a version — you can rollback later from the project's History panel.
