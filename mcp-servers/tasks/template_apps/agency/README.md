# Agency Template

A bold, design-forward studio site template for creative agencies. Dark charcoal background (`#0a0a0b`), off-white text, and a lime-electric accent (`#c1ff00`). Features a mouse-tracking cursor dot, infinite-scroll marquee strip, animated gradient headline, count-up statistics, and a simulated contact form with toast feedback. Uses Inter for body copy and Space Grotesk for display headings.
