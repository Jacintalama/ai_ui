# Event Template

A bold, modern conference / festival landing page built with Alpine.js, Tailwind CSS (CDN), and Space Grotesk. The template features a live countdown to the event date, scroll-triggered count-up stats, a 12-speaker grid with hover-bio reveal (keyboard-accessible), Day 1 / Day 2 schedule tabs, a tiered sponsor section, venue photos, three ticket tiers, and a FAQ accordion — all on a deep navy background with neon cyan accents.
