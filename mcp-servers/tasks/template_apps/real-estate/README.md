# Real Estate Template

Editorial property listing page for a restored Victorian home in Brookline, MA. Features an auto-advancing hero carousel, animated count-up stats, a nine-image masonry gallery with lightbox, an interactive neighborhood map placeholder, and a "Schedule a Viewing" form that dispatches a toast on submit. Palette: cream `#faf7f2` / slate `#1f2937` / warm gold `#b08a3e`. Typography: Cormorant Garamond (display) + Inter (body).
