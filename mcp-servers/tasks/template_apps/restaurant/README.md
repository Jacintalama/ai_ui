# Restaurant Template

A warm, atmospheric single-page restaurant website for a wood-fired Italian trattoria. The design uses a cream-and-espresso palette with terracotta accents, Playfair Display for headings and Inter for body copy, and features a parallax hero, a 4-tab Alpine-powered menu (Brunch, Lunch, Dinner, Drinks), a CSS masonry photo gallery, an hours/location section with a styled map placeholder, a reservation form with toast feedback, and a rich footer with address and social links.
