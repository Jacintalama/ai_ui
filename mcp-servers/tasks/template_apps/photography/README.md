# Photography Template

Image-led portfolio template for independent photographers. Pure black canvas (#000000), white typography with wide letter-spacing, and a masonry gallery with a keyboard-navigable lightbox. Built with Alpine.js, Tailwind CSS, and vanilla IntersectionObserver for scroll-reveal animations.
