Features

Everything you need, nothing you don't

CRUDSimple gives your team a clean interface to manage Supabase data without writing a single line of backend code.

One-click Supabase connect

Paste your Supabase URL and anon key. CRUDSimple introspects your schema automatically — no manual table setup.

Full CRUD out of the box

Create, read, update, and delete rows across any table. Inline editing, bulk delete, and optimistic UI updates included.

Instant search & filter

Filter by any column, sort ascending or descending, and search across text fields — all without writing a single SQL query.

Row-Level Security aware

CRUDSimple respects your existing Supabase RLS policies. Your data stays protected — no admin backdoors, no bypasses.

Team access controls

Invite teammates with viewer or editor roles. Keep sensitive tables read-only for junior members while admins get full write access.

CSV export in one click

Export any table or filtered view to CSV instantly. Share data with stakeholders or import it into spreadsheets without touching Supabase Studio.
