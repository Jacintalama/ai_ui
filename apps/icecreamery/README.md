# The Icecreamery

A static marketing website for an artisan ice cream shop featuring a full flavour menu, seasonal specials, an About section, a table reservation form, and a contact form. Built with plain HTML5, Tailwind CSS (CDN), Alpine.js, and vanilla ES modules — no build step required.

## How to run

Open `index.html` directly in any modern browser, or serve the folder with any static file server:

```bash
npx serve apps/icecreamery
# or
python3 -m http.server 8080 --directory apps/icecreamery
```
