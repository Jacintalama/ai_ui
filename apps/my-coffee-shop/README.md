# Grounds & Grace Coffee Shop

A responsive, static website for a specialty coffee shop featuring a full drinks menu with pricing and descriptions, an Our Story section, seasonal specials, hours and location info, and a contact form — all built with Tailwind CDN, Alpine.js, and vanilla ES modules (no build step required).

## How to run

Open `apps/my-coffee-shop/index.html` in any modern browser, or serve the folder with any static file server:

```bash
npx serve apps/my-coffee-shop
# or
python3 -m http.server 8080 --directory apps/my-coffee-shop
```
